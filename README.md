# ScrollView-Test

## 本demo所要实现的效果如下
    1  为了在滑动Scrollview 或者是其子类的时候能将上面的搜索框自动隐藏<br>
    2  用户向上滑动Scrollview时显示搜索框<br>
    3  用户向下滑动Scrollview时隐藏搜索框<br>
